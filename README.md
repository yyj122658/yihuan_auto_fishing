# 异环自动钓鱼工具

这是一个 Windows 下使用的《异环》自动钓鱼脚本。脚本通过识别游戏画面中的钓鱼提示、拉条和结算界面，自动完成抛竿、上鱼、拉条和跳过结算流程。

## 功能

- 自动查找标题包含“异环”的游戏窗口
- `F8` 开始或暂停自动钓鱼
- `F9` 手动强制按一次 `F`，并进入等待上鱼状态
- `F10` 退出脚本
- 识别右下角 `F` 按钮蓝色提示，自动上鱼
- 识别绿色拉条和黄色光标，自动按 `A` / `D` 控制拉条
- 结算阶段自动点击或按空格跳过
- 等待上鱼超时后自动点击恢复并重新抛竿

## 重要限制

当前版本依赖屏幕截图和前台输入：

- 游戏窗口需要保持可见，不能被浏览器或其他窗口遮挡关键区域。
- 脚本会在需要操作时把游戏窗口切到前台。
- 如果游戏最小化、被遮挡，或你在前台操作其他窗口，识别和输入都可能失败。
- 目前不支持稳定的完全后台钓鱼。

如果想边看视频边运行，建议使用分屏、第二显示器，或让游戏窗口和浏览器窗口互不遮挡。

## 运行可执行文件

仓库中如果存在打包后的文件，可以直接运行：

```powershell
.\dist\auto_yihuan_fishing_stable.exe
```

由于打包配置启用了管理员权限，运行时可能会弹出 UAC 提示。

## 从源码运行

需要 Python 3.10+，并安装依赖：

```powershell
pip install opencv-python numpy mss pydirectinput pywin32
```

运行脚本：

```powershell
python .\auto_yihuan_fishing_stable.py
```

## 使用步骤

1. 启动游戏，并进入可以钓鱼的场景。
2. 确保游戏窗口标题包含“异环”。
3. 启动脚本或 exe。
4. 按 `F8` 开始自动钓鱼。
5. 需要暂停时再次按 `F8`。
6. 需要退出时按 `F10`。

## 重新打包


打包后的程序会生成在 `dist\` 目录中。

## 常见问题

### 找不到游戏窗口

确认游戏已经打开，并且窗口标题包含脚本中的配置：

```python
GAME_WINDOW_TITLE = "异环"
```

如果实际窗口标题不同，需要修改这个值。

### 脚本没有按键效果

尝试以管理员身份运行脚本或 exe。部分游戏进程权限较高时，普通权限的脚本可能无法发送输入。

### 识别不到上鱼或拉条

可以在脚本中临时打开调试窗口：

```python
DEBUG_VIEW = True
```

调试完成后建议改回 `False`。

### 浏览器盖住游戏后失效

这是当前实现的正常限制。脚本使用 `mss` 截取屏幕坐标，遮挡后截到的是遮挡窗口，不是游戏画面。
